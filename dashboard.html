Telegram Editorial Bot

Every Monday the bot sends 20 article ideas per site to your Telegram. Reply with numbers to publish, or "Auto" to let it decide.

Free & serverless. Runs on your Mac, a Raspberry Pi, or Railway.app free tier.
🔧 One-Time Setup
1
Message @BotFather on Telegram → /newbot → copy the token.
2
Send your bot any message, then visit:
https://api.telegram.org/bot<TOKEN>/getUpdates → copy your chat_id.
3
Add secrets: TELEGRAM_BOT_TOKEN and TELEGRAM_CHAT_ID to your bot repo's GitHub Secrets.
4
Run locally: 
cd /path/to/content-network
python telegram_bot.py
📋 How the Workflow Works
📅
Monday 9am UTC
Bot generates 20 ideas per site and sends numbered list to Telegram
✍️
You reply
Type 1 3 7 12 to pick articles, or Auto to publish the 7 best
🚀
Auto-publish
Articles generated, pushed to GitHub, newsletters sent, confirmation with links

Newsletter Automation

Brevo (free up to 300 emails/day, 9,000/month) — perfect for a 20-site network.

Free tier math: 20 sites × 100 subscribers × 1 weekly email = 2,000 emails/week = well within the free limit.
📋 Setup (one-time)
1
Create a free account at brevo.com
2
Settings → API Keys → Generate → add as BREVO_API_KEY in GitHub Secrets
3
Contacts → Lists → Create one list per site. Note the List ID from the URL.
4
In Launcher → Step 3 → each site → set the Brevo List ID field.
5
Add a Brevo subscribe form to each site (embed code in Cloudflare Pages). One email field, submit = instant subscribe.
⚡ What Happens Automatically
Every time an article publishes, bot.py automatically:
1. Builds a branded email in that site's theme colors
2. Creates a campaign in Brevo via API
3. Sends it immediately to that site's subscriber list
4. Each site's email looks completely different — branded to the site's design

GitHub Setup

Everything you need to get all 20 sites live on Cloudflare Pages for free.

🔑 GitHub Token (for this dashboard)
Stored in localStorage only — never sent anywhere except api.github.com
☁ Cloudflare Domain Manager
Connect domains directly from this dashboard using the Cloudflare API. Checks DNS status and shows which domains are active on Cloudflare Pages.
Get your API Token: Cloudflare dashboard → My Profile → API Tokens → Create Token → use "Edit zone DNS" template. Account ID is in the right sidebar of the Cloudflare home page.
📦 Per-Site Setup (repeat for all 20 sites)
1
Create a public GitHub repo for each site — e.g. yourusername/cryptoinsights
2
Cloudflare Pages: Dashboard → Pages → Create Project → Connect to Git → select the repo. Framework: None. Build: empty. Output: /
3
Custom domain: Pages → Custom Domains → add domain → update nameservers to Cloudflare. Auto SSL included. Free.
4
Bot repo: Create one more repo for the bot itself (bot.py, telegram_bot.py, network-config.json, .github/workflows/)
5
GitHub Secrets in the bot repo: ANTHROPIC_API_KEY, PEXELS_API_KEY, BREVO_API_KEY, TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID. GITHUB_TOKEN is auto-provided.
⚡ Workflows
daily-post.yml
8am UTC daily. Publishes one article per site. Fully automatic.
weekly-ideas.yml
Monday 9am UTC. Sends 20 ideas to Telegram for your approval.
